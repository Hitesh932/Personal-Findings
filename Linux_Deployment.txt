Step:3
---------

Download postgres Sql 

Install the repository RPM:
--> sudo dnf install -y https://download.postgresql.org/pub/repos/yum/reporpms/EL-9-x86_64/pgdg-redhat-repo-latest.noarch.rpm

Disable the built-in PostgreSQL module:
--> sudo dnf -qy module disable postgresql

Install PostgreSQL:
--> sudo dnf install -y postgresql15-server

Optionally initialize the database and enable automatic start:
--> sudo /usr/pgsql-15/bin/postgresql-15-setup initdb
--> sudo systemctl enable postgresql-15
--> sudo systemctl start postgresql-15

Step:4 
---------
Access PostgreSQL Prompt: Use the following command to access the PostgreSQL prompt as the default PostgreSQL superuser (postgres):

--> sudo -i -u postgres

Then set the postgres user password 

--> \password postgres

it will ask for password and enter the password and note it down.

Step:5
---------
Create a Database
Inside the PostgreSQL prompt (psql), you can create a new database using SQL commands. Here’s an example of creating a database named mydatabase:

--> CREATE DATABASE mydatabase;

Replace mydatabase with your desired database name. Make sure to end SQL statements with a semicolon (;).

Step:6
---------
Create a User and Grant Permissions
Next, you can create a new PostgreSQL user and grant it permissions on the database. Here’s an example of creating a user named myuser with a password and granting it all privileges on the mydatabase:

--> CREATE USER myuser WITH PASSWORD 'password';

--> GRANT ALL PRIVILEGES ON DATABASE mydatabase TO myuser;

Replace myuser with your desired username and 'password' with your desired password. Remember to enclose the password in single quotes ('').

Step:7
-------
Exit PostgreSQL Prompt
Once you have created the database and user, you can exit the PostgreSQL prompt by typing:

--> \q

This command will return you to the regular Linux shell prompt.
